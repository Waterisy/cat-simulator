# 🐱 养猫模拟器 Pro

一个功能丰富的 2D 网页养猫游戏，无需安装，打开浏览器即可游玩！

## 🎮 游戏玩法

- **每 15 秒**自动获得金币收益，猫越多收益越高
- 领养小猫时随机获得品种，越稀有的品种售价越高
- 给猫咪喂食和玩耍，帮助它们成长
- 小猫会随机触发各种**事件**，影响成长速度和品质
- 幼猫 → 小猫 → 成猫，成猫可以出售获得金币
- 猫咪的**品种 × 品质 × 市场价**决定最终售价！
- 在**猫咪股市**低买高卖赚取差价

## 📈 猫咪股市

- 每个品种的成猫都有**独立的市场价格**，每天随机波动
- 稀有品种价格波动幅度更大，风险与收益并存
- 可在**低价时**从股市直接购入成猫，**高价时**出售获利
- 股市页面提供 **7 日走势迷你图**，辅助判断买卖时机
- 最终售价 = 市场价 × 品质倍率

## 🤖 自动托管系统

在「托管」Tab 可开启全自动运营，支持精细化策略配置：

| 功能 | 说明 |
|------|------|
| 🐟 自动喂食 | 饱食度低于阈值时自动使用猫粮 |
| 🧶 自动玩耍 | 心情低于阈值时自动玩耍 |
| 💊 自动治病 | 猫咪生病时自动使用药品（优先级最高） |
| 🍖 自动营养 | 有库存时自动喂营养餐提升品质 |
| 🏪 自动出售 | 成猫品质达到设定阈值时自动出售 |
| 📈 自动股市买入 | 价格低于基准价 N% 时自动买入 |
| 📉 自动股市卖出 | 价格高于买入价 N% 时自动卖出 |

所有阈值均可通过滑块自由调节，实现挂机放置流玩法！
