# 🐱 养猫模拟器 Pro

一个功能丰富的 2D 网页养猫游戏，无需安装，打开浏览器即可游玩！

> 纯 HTML5 + CSS3 + Vanilla JavaScript，单文件，零依赖

## 🚀 如何运行

直接用浏览器打开 `index.html` 即可，或访问在线版：

👉 **[GitHub Pages 在线游玩](https://waterisy.github.io/cat-simulator/)**

---

## 🎮 游戏玩法概览

1. **注册/登录**账号，游戏进度自动保存到浏览器本地
2. 花 💰50 **领养小猫**，随机获得品种
3. 给猫咪**喂食、玩耍**，帮助它们成长
4. 幼猫 → 小猫 → 成猫，成猫可以**出售**换金币
5. 关注**猫咪股市**，低买高卖赚取差价
6. 开启**自动托管**，挂机放置也能赚钱！

## 👤 用户系统

| 功能 | 说明 |
|------|------|
| 📝 注册 | 用户名（≥2位）+ 密码（≥6位），本地创建账号 |
| 🔑 登录 | 验证通过后自动读取该账号存档 |
| 💾 自动保存 | 每 30 秒自动保存；买卖等关键操作后 3 秒内快速保存 |
| 👥 多账号 | 每个账号独立存档，互不干扰 |
| 🚪 退出登录 | 保存后退出，下次登录继续 |

> ⚠️ 数据存储在浏览器 `localStorage`，清除浏览器缓存会丢失数据。

## 📈 猫咪股市

- 每个品种成猫都有**独立的市场价格**，每天随机波动
- 稀有品种价格波动幅度更大，风险与收益并存
- 在**低价时**从股市直接购入成猫（无需养成），**高价时**出售获利
- 页面提供 **7 日走势迷你图**，辅助判断买卖时机
- 点击「⏭ 跳过一天」推进行情

## 🤖 自动托管系统

在「托管」Tab 开启全自动运营，支持精细化策略：

| 功能 | 说明 |
|------|------|
| 🐟 自动喂食 | 饱食度低于阈值时自动使用猫粮 |
| 🧶 自动玩耍 | 心情低于阈值时自动玩耍 |
| 💊 自动治病 | 猫咪生病时自动使用药品（优先级最高） |
| 🍖 自动营养 | 有库存时自动喂营养餐提升品质 |
| 🏪 自动出售 | 成猫品质达到设定阈值时自动出售 |
| 📈 自动股市买入 | 价格低于基准价 N% 时自动买入 |
| 📉 自动股市卖出 | 价格高于买入价 N% 时自动卖出 |

所有阈值均可通过滑块自由调节，实现挂机放置流玩法！
