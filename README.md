# 🐱 养猫模拟器

一个可爱的 2D 网页养猫游戏，无需安装，打开浏览器即可游玩！

## 🎮 游戏玩法

- **每 15 秒**自动获得金币收益，猫越多收益越高
- 用金币在商店购买小猫、猫粮、玩具
- 给猫咪喂食和玩耍，帮助它们成长
- 小猫 → 幼猫 → 成猫，成猫可以出售获得更多金币
- 购买扩大猫舍可以养更多猫！

## 🛒 商品

| 物品 | 价格 | 说明 |
|------|------|------|
| 🐱 小猫咪 | 50 💰 | 购买一只新猫咪 |
| 🐟 猫粮 x1 | 15 💰 | 补充饱食度 + 促进成长 |
| 🐟 猫粮 x5 | 60 💰 | 批量购买更划算 |
| 🧶 玩具球 | 30 💰 | 提升心情 + 促进成长 |
| 🏡 扩大猫舍 | 200 💰 | 猫舍容量 +2 |

## 🚀 如何运行

直接用浏览器打开 `index.html` 即可，无需服务器！

或者访问：[GitHub Pages](https://waterisy.github.io/cat-simulator/)

## 技术栈

- 纯 HTML5 + CSS3 + Vanilla JavaScript
- Canvas 2D 绘制猫咪
- 无任何外部依赖
