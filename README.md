# 🐱 养猫模拟器

一个可爱的 2D 网页养猫游戏，无需安装，打开浏览器即可游玩！

## 🎮 游戏玩法

- **每 15 秒**自动获得金币收益，猫越多收益越高
- 用金币在商店购买小猫、猫粮、玩具等道具
- 给猫咪喂食和玩耍，帮助它们成长
- 小猫会随机触发各种**事件**，影响成长速度和品质
- 幼猫 → 小猫 → 成猫，成猫可以出售获得金币
- 猫咪的**品质**越高，出售价格越高！
- 购买扩大猫舍可以养更多猫！

## 🎭 事件系统

猫咪每隔一段时间会随机触发事件，触发时会弹出通知弹窗：

| 事件 | 类型 | 效果 |
|------|------|------|
| 😊 心情大好 | 积极 | 成长速度×1.5，品质+3 |
| ⚡ 精力充沛 | 积极 | 成长速度×2.0，品质+4 |
| 🍀 幸运降临 | 积极 | 饱食度/心情+20，品质+5 |
| 💫 灵感爆发 | 积极 | 成长速度×2.5，品质+6 |
| 🔍 好奇探索 | 中性 | 心情+15，饱食度-10，品质+1 |
| 😴 犯困了   | 中性 | 成长速度×0.5 |
| 🤒 生病了   | 负面 | 成长**停止**，需使用💊药品治愈 |
| 😿 情绪低落 | 负面 | 心情-25，成长速度×0.6 |

> 积极事件会给猫咪**永久提升品质**属性！

## ⭐ 品质系统

品质决定成猫的最终售价，可通过积极事件和营养餐提升：

| 品质 | 等级 | 售价倍率 |
|------|------|----------|
| 0~19   | ☆☆☆☆☆ 普通 | ×1.0 |
| 20~39  | ★☆☆☆☆ 良好 | ×1.3 |
| 40~59  | ★★☆☆☆ 优秀 | ×1.7 |
| 60~79  | ★★★☆☆ 精良 | ×2.3 |
| 80~94  | ★★★★☆ 稀有 | ×3.2 |
| 95~100 | ★★★★★ 传说 | ×5.0 |

## 🚀 如何运行

直接用浏览器打开 `index.html` 即可，无需服务器！

或者访问：[GitHub Pages](https://waterisy.github.io/cat-simulator/)

## 技术栈

- Canvas 2D 绘制猫咪
- 无任何外部依赖
